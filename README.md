## 特性

- 使用Flutter跨平台框架开发
- Material3主题
- 适配Pad UI
- 日程课表
- 成绩查询
- 全校课表
- 社会成绩查询
- 空教室
- 慧生活798
- 无任何广告
- 更多特性开发中

## 安装

访问本项目的 [Releases](https://github.com/YiQiuYes/schedule/releases)
页面下载安装包。

- Android 用户可以直接通过 `apk` 格式文件安装

- IOS 本安装包无签名，用户下载 `ipa` 格式安装包后，通过 [TrollStore](https://github.com/opa334/TrollStore) 签名或者自签安装
